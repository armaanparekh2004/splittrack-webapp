YOUR DATA
Local storage
Where this lives

Everything you enter is saved automatically to this browser's local storage — nothing leaves your machine. That means it won't sync to your phone yet, and clearing your browser data will erase it. Export a backup regularly, especially before clearing your browser cache or switching computers.

Import from Chase

Export a CSV from chase.com → your account → Activity → Download, then import it here. New transactions merge with what's already saved — nothing is duplicated or overwritten.

AI categorization
Checking…

Browsers can't call Anthropic's API directly, so categorization goes through a small serverless function deployed alongside this page — it holds your API key server-side and is the only thing that talks to Anthropic. If you deployed this app and its /api/categorize function together on Vercel, this works automatically with no setup below. Running this page locally as a plain file instead, or hosting the API elsewhere? Paste that URL here to override.

Backup & restore
Relink old splits

Splits created before this feature existed show whatever name the transaction had at the moment you split it, instead of staying in sync if you rename it later. This finds those old splits and links them to their real transaction — matched only by exact date plus the person already being on that transaction's split list, so nothing gets linked unless it's unambiguous. Anything it can't match confidently with certainty is left alone, not guessed.
